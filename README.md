# exam-web-project

# Экзаменационный проект по дисциплине «Основы Веб-Технологий»

**Демо-сайт**: [https://mariguss.github.io/exam-web-project/](https://mariguss.github.io/exam-web-project/)

Проект представляет собой многостраничный веб-сайт языковой школы **LinguaSchool**, реализующий функционал просмотра курсов и репетиторов, подачи заявок на обучение, а также управления заявками в личном кабинете.

## Основные возможности

### Главная страница
- Отображение списка курсов (с пагинацией, до 5 на странице)
- Отображение таблицы репетиторов
- Поиск курсов по названию и уровню
- Фильтрация репетиторов по языку и опыту
- Возможность оформить заявку на курс или репетитора

### Расчёт стоимости
- Автоматический расчёт цены с учётом:
  - Базовой ставки (курс/репетитор)
  - Количества студентов (1–20)
  - Выходных дней (+50%)
  - Времени занятия (утро +400 ₽, вечер +1000 ₽)
  - Дополнительных опций (доп. материалы, индивидуальные занятия, экскурсии и др.)
  - Скидок (ранняя регистрация –10%, групповая запись –15%)

### Личный кабинет
- Просмотр всех заявок с пагинацией
- Подробная информация о заявке («Подробнее»)
- Редактирование заявки (включая изменение продолжительности у репетитора от 1 до 40 часов)
- Удаление заявок
- Адаптивный интерфейс с цветными кнопками действий

### Интерактивная карта
- Интеграция Яндекс.Карт
- Отображение языковых ресурсов Москвы:
  - Библиотеки
  - Культурные центры
  - Языковые кафе и клубы
  - Частные курсы
- Фильтрация по типам объектов

## Технологии
- HTML5, CSS3, Bootstrap 5
- JavaScript (ES6+, async/await, fetch API)
- Яндекс.Карты API
- GitHub Pages (хостинг)
- Статическая архитектура (без серверной части)
